Step 3: Set Base Styles on the <body> Tag

  • Create a new css file called "styles.css" in the same folder as your HTML file.
    Note This file is already linked to our HTML skeleton.
  • Set the font family to Helvetica Neue with an Arial fallback and generic family of sans serif
  • Set the font size to 12 pixels
  • Set the leading/line height to 18 pixels
  • Set the background color to light gray (#CCCCCC)
  • Remove margins
  • Add 20 pixels of padding to the top and bottom, remove left and right padding

Step 3: Solution

body {
	font-family: 'Helvetia Neue', Arial, sans-serif;
	font-size: 12px;
	line-height: 18px;
	background-color: #CCCCCC;
	margin: 0;
	padding-top: 20px;
	padding-bottom: 20px;
	padding-left: 0;
	padding-right: 0;
}

Step 4: Center the Layout

Lets center our HTML page by placing some styles on the container div.

  • Create a new CSS selector for the div with an id of container
  • Set the width to 960 pixels (assuming we are targeting a minimum resolution of 1024x768)
  • Set the left and right margins to 0 to center the container
  • Set the background color to white (#FFFFFF)

Step 4: Solution

#container {
	width: 960px;
	margin-left: auto;
	margin-right: auto;
	background-color: #FFFFFF;	
}

Step 5: Masthead, Logo and Tagline Styles

Lets give the masthead area a background color. We also want to position the logo and tagline's absolute so we can specify exactly where they should display.

  • Write a new CSS selector for an element with an id of masthead
  • Set the background color to dark gray (#333333)
  • Write a new CSS selector for an element with an id of logo
  • Set the logo to be positioned absolutely
  • Set the logo to be 50 pixels from the left and 50 pixels from the top
  • Set the tagline to be positioned absolutely
  • Set the tagline to be 200 pixels from the right and 50 pixels from the top

Step 5: Solution

#masthead {
	background-color: #333333;
}

#logo {
	position: absolute;
	left: 50px;
	top: 50px;
}

#tagline {
	position: absolute;
	right: 200px;
	top: 50px;
}

Step 6: Fix the masthead

Uh oh! The logo and tagline are being positioned based on the body tag. We need them to be positioned based on the masthead.

  • Update the masthead selector to set the position to relative so the logo and tagline will be positioned based on it

Step 6: Solution

#masthead {
	background-color: #333333;
	position: relative;
}

Step 7: Fix the masthead... more

So now the tagline and logo are positioned based on the masthead, but where is the masthead background color? Because the tagline and logo are absolute positioned they dont take up any space in the masthead. We need to prop the masthead open by declaring a height.

  • Update the masthead selector to set the height to 150 pixels

Step 7: Solution

#masthead {
	background-color: #333333;
	position: relative;
	height: 150px;
}

Step 8: Logo and tagline styles

Lets also give the logo and tagline some font styles. We will also add some styles to the link in the logo.

  • Update the logo to set the font size to 18 pixels and be bold
  • Update the tagline to set the font size to 14 pixels, italic and white (#FFFFFF)
  • Create a new CSS Selector for anchor tags inside the logo div and remove the underline
  • Create a new CSS Selector the unvisited link state in the logo div and set the color to white
  • Create a new CSS Selector the visited link state in the logo div and set the color to light gray (#CCCCCC)
  • Create a new CSS Selector the hover link state in the logo div and set the color to red
  • Create a new CSS Selector the active link state in the logo div and set the color to pink

Step 8: Solution

#logo {
	position: absolute;
	left: 50px;
	top: 50px;
	font-size: 18px;
	font-weight: bold;
}

#logo a {
	text-decoration: none;
}

#logo a:link {
	color: #FFFFFF;
}

#logo a:visited {
	color: #CCCCCC;
}

#logo a:hover {
	color: red;
}

#logo a:active {
	color: pink;
}

#tagline {
	position: absolute;
	right: 200px;
	top: 50px;
	font-size: 14px;
	font-style: italic;
	color: #FFF;
}

Step 9: 2 Column Layout

We want to create a 2 column layout with the navigation as a smaller left column and the content as a larger right column. To achieve this we will float both divs and give them appropriate widths.

  • Write a new CSS selector for the div with the id navigation
  • Set the navigation to float left
  • Set the width of the navigation to 200 pixels
  • Set the content to float left
  • Set the width of the content to 700 pixels
  • Set the left, top and bottom (not right) padding on the content div to 20 pixels

Step 9: Solution

#navigation {
	float: left;
	width: 200px;
}

#content {
	float: left;
	width: 700px;
	padding-left: 20px;
	padding-top: 20px;
	padding-bottom: 20px;
}

Step 10: Fix the footer

The copyright notice from our footer is appearing over to the right of the content div. To fix this, we need to clear our float.

  • Write a CSS selector for the footer div
  • Set clear to left to clear our floats

Step 10: Solution

#footer {
	clear: left;
}

Step 11: Navigation Styles

Lets remove those bullets from our navigation. While we are at it, lets make our navigation links easier to click.

  • Write a CSS selector for the unordered list inside of the navigation div
  • Set the list style type to none
  • Remove all margins from the list
  • Remove all padding from the list
  • Write a CSS selector for the anchor links inside of the navigation div
  • Set the links to display block
  • Add 10 pixels of padding to the links
  • Remove the underlines from the link
  • Set the background color to light gray (#CCCCCC)
  • Set the color to black (#000000)
  • Add a 1 pixel solid dark gray border to the links (#333333)

Step 11: Solution

#navigation ul {
	list-style-type: none;
	margin: 0;
	padding: 0;
}

#navigation a {
	display: block;
	padding: 10px;
	text-decoration: none;	
	background-color: #CCCCCC;
	color: #000000;
	border-bottom: 1px solid #333333;
}

Step 12: Navigation Rollover Styles

Lets reverse the background and text colors of the navigation links when you rollover them.

  • Write a CSS selector for the hover state of links in the navigation div
  • Set the text color to white (#FFFFFF)
  • Set the background color to dark gray (#666666)

Step 12: Solution

#navigation a:hover {
	background-color: #666666;
	color: #FFFFFF;
}

Step 13: Footer Styles

Lets center the text in our footer and make sure it doesnt touch the edges of the container. We will also set the background color to a medium gray.

  • Update the footer CSS selector we created.
  • Set the text alignment to center
  • Add 30 pixels of padding to all sides
  • Set the background color to medium gray (#999999)

Step 13: Solution

#footer {
	clear: left;	
	text-align: center;
	padding: 30px;
	background: #999999;
}

Step 14: Footer Link Styles

Finally, lets update the default blue links in our footer.

  • Create a new CSS Selector the unvisited link state in the footer div and set the color to black (#000000)
  • Create a new CSS Selector the visited link state in the footer div and set the color to dark gray (#333333)
  • Create a new CSS Selector the hover link state in the footer div and set the color to black and remove the underline
  • Create a new CSS Selector the active link state in the footer div and set the color to red and remove the underline

Step 14: Solution

#footer a:link {
	color: #000000;
}

#footer a:visited {
	color: #333333;
}

#footer a:hover {
	color: #000000;
	text-decoration: none;
}

#footer a:active {
	color: red;
	text-decoration: none;
}
