  • Write a css selector to set the font size on the body tag to 12 pixels and the leading/line height to 18 pixels.
  • Note: The CSS examples in this slideshow display only the CSS for the current step. Each step involves adding more CSS to the styles.css, not replacing the content.

Step 3: Link Your HTML & CSS

  • Add a link tag to the head of your HTML document to attach the stylesheet you created.
  • Use the href attribute to specify to url of the stylesheet (styles.css)
  • Set the rel attribute of the link tag to "stylesheet" to let the browser know the relevance of the linked document is that its a stylesheet.
  • Set the type attribute of the link tag to "text/css" to let the browser know type of stylesheet it is.
  • Set the media attribute of the link tag to "all" to specify that the stylesheet should be used for print, screen, mobile, etc.

Step 3: Solution

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
	<head>
		<title>Glassman Benjamin | GDD 410 Layout 1 Assignment</title>
		<link href="styles.css" rel="stylesheet" type="text/css" media="all" />
	</head>
	<body>
	
	</body>
</html>

Step 6: Add Descriptive Classes

Our HTML is very generic right now, which will make it harder to style effectively. Lets add some descriptive classes to our blog post to use as hooks for our CSS

  • Add a class of "post" to the div holding the post (Note: Not the div with the id of "container")
  • Add a class of "title" to the heading with the blog post title
  • Add a class of "date-author" to the paragraph holding the date posted and author
  • Add a class of "teaser" to the paragraph holding the teaser
  • Add a class of "read-more" to the paragraph holding the read more link

Step 6: Solution

			<h1>My Blog</h1>
			<div class="post">
				<h2 class="title">Blog Post 1</h2>

Step 8: Post Separators

Lets apply some styles to our posts. This step occurrs in our "styles.css" file. Lets add a gray border to the bottom of each post to separate them. Lets also add 20 pixels of padding to the top and bottom of each post.

  • Write a new CSS selector that applies to each post.
    Hint: The Class Selector is a period followed by the class name (.class)
  • Set the bottom border to 1 pixel, solid and gray (#999999)
  • Set the top and bottom padding to 20 pixels

Step 8: Solution

.post {
	padding-top: 20px;
	padding-bottom: 20px;
	border-bottom: 1px solid #999999;
}

Step 9: Post Title Styling

Lets apply some styles to our post headings. This step occurrs in our "styles.css" file. Remember, the title has a class of "title" and is inside of a div with a class of "post".

  • Write a new CSS selector that applies to anything with a class of "title" inside anything with a class of "post".
    Hint: You can separate selectors with a space to combine them (.class1 .class2 would apply to tags with "class2" that are inside tags with "class1")
  • Set the font family to Helvetica Neue, with a fallback to Arial and a generic family of sans serif.
    Hint: You can separate fonts with commas to declare a series of fallback fonts called a font stack. The final item in the list is a generic family. Font names with spaces in them must be surrounded in single quotes.
  • Set the font size to 18 pixels
  • Set the leading/line height to 24 pixels
  • Remove bolding from the heading
  • Set the color to dark gray (#666666)
  • Set the top margin to 0
  • Set the bottom margin to 9 pixels

Step 9: Solution

.post .title {
	font-family: 'Helvetica Neue', Arial, sans-serif;
	font-size: 18px;
	line-height: 24px;
	font-weight: normal;
	color: #666666;
	margin-top: 0;
	margin-bottom: 9px;
}

Step 10: Date & Author Styling

Lets make the date and author lighter gray and italic.

  • Write a CSS selector for the "date-author" class inside the "post" class.
  • Remove all margins
  • Set the color to medium gray (#999999)
  • Italicize the text

Step 10: Solution

.post .date-author {
	margin: 0;
	color: #999999;
	font-style: italic;
}

Step 11: Teaser Styling

Lets increase the font size for the teaser.

  • Write a CSS selector for the "teaser" class inside the "post" class.
  • Set the font size to 13 pixels
  • Remove all margins

Step 11: Solution

.post .teaser {
	font-size: 13px;
	margin: 0;
}

Step 12: Read More Styling

Lets set the color of the read more link to black with a red rollover.

  • Remove margin from the read more paragraph
  • Make the read more links black (#000000) before the user has visited them
  • Make the read more links dark gray (#333333) after the user has visited them
  • Make the read more links red when hovered
  • Make the read more links pink when active

Step 12: Solution

.post .read-more {
	margin: 0;
}

.post .read-more a:link {
	color: #000000;
}

.post .read-more a:visited {
	color: #333333;
}

.post .read-more a:hover {
	color: red;
}

.post .read-more a:active {
	color: pink;
}

Step 13: Layout

Lets center our container by giving it a width and left/right margins of auto. We will also give it a white background and padding. Lets change the body background to dark gray.

  • Set the background color of the body tag to gray (#666666)
    Hint: Add this to your existing css selector for the body tag at the top of styles.css
  • Set the width of the container div to 960 pixels
  • Center the container div horizontally by setting left and right margins to auto
  • Set the background color of the container div to white (#FFFFFF)
  • Set padding on all sides of the container div to 36 pixels

Step 13: Solution

body {
	background-color: #000000;
}

#container {
	width: 960px;
	margin-left: auto;
	margin-right: auto;
	background-color: #FFFFFF;
	padding: 36px;
}
