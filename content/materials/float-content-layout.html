Common Content Layout Patterns using Float

1. Text flows around image

This is the most common use of floats for basic content layout. You have a few paragraphs of content on a page and you add an image. You want the text to flow around the image.

Solution

Add an HTML wrapper to the image (paragraph or div tag) with an image-right or image-left class. Also add bottom margin and left or right margin to add space around the image. Hint: You must place the image before the content you want to flow around it.

Example

CSS

.image-right {
    float: right;
    margin-left: 20px;
    margin-bottom: 10px;
}

.clear {
    clear: both;
}

HTML

<div class="image-right">
    <img src="images/diving-dog-1-620x353.png" width="300" alt="Dog Diving Underwater" />
</div>
<p>Lorem ipsum dolor sit amet, consectetur adipiscing etc. etc.</p>
<p>Lorem ipsum dolor sit amet, consectetur adipiscing etc. etc.</p>
<p>Lorem ipsum dolor sit amet, consectetur adipiscing etc. etc.</p>
<div class="clear"></div><-- Clearing div to prevent further content from flowing around floated image -->

2. Text Column next to Image Column (or any 2 columns)

Often times you want to create a 2 column layout where the text in column a does not flow around the image in the other column.

Solution

Create a wrapping tag (article or div tag) for each column, giving both tags a unique class. In our example we will show an employee listing from a staff page. We will have a wrapping tag for a headshot image and a wrapping tag for the bio. You can achieve this by floating the first wrapper and then setting a margin on the second one. If you float the headshot to the right and it is 200 pixels wide, then set 200 pixels of right margin on the bio so it will not flow around the headshot. If you float the headshot to the left, then use left margin on the bio. Hint: When you float an HTML element it will shrinkwrap to fit the width of the content. Sometimes you need to set a width to constrain it to a certain size (especially when the column contains an unknown amount of text or has a specific desired width).

Example

CSS

.headshot {
    float: left;
    width: 300px;
}

.bio {
    margin-left: 330px; /* Extra 30px of margin to prevent the bio from the touching the headshot */
}

.clear {
    clear: both;
}

HTML

<div class="employee">
    <div class="headshot">
        <img src="images/diving-dog-1-620x353.png" width="300" alt="Dog Diving Underwater" />
    </div>
    <div class="bio">
        <p>Lorem ipsum dolor sit amet, etc. etc.</p>
        <p>Duis condimentum semper diam, etc. etc.</p>
    </div>
    <div class="clear"></div><-- Clearing div to ensure employee div wraps around floated content -->
</div>

3. Grid of thumbnails in rows

Sometimes your designs will call for a grid display of rows and columns. For example, a photo gallery often displays thumbnails in a grid with each row containing a certain number of thumbnails.

Solution

We can achieve this grid layout using wrapper tags (div tags generally) for each row and multiple floated html tags within each row. Each row will include a clearing div to ensure it wraps around the items in the row and will clear any past floats.

Example

CSS

.thumbnail-row {
    clear: both;
    border-bottom: 1px solid #ccc;
    padding-bottom: 15px; /* Padding adds spacing between the border and the thumbnails */
    margin-bottom: 15px; /* Margin adds spacing between the rows */
}

.thumbnail {
    float: left;
    width: 150px;
    margin-right: 15px;
}

.clear {
    clear: both;
}

HTML

<div class="thumbnail-row">
    <div class="thumbnail">
        <img src="images/diving-dog-1-620x353.png" width="150" alt="Dog Diving Underwater" />
    </div>
    <div class="thumbnail">
        <img src="images/diving-dog-1-620x353.png" width="150" alt="Dog Diving Underwater" />
    </div>
    <div class="thumbnail">
        <img src="images/diving-dog-1-620x353.png" width="150" alt="Dog Diving Underwater" />
    </div>
    <div class="clear"></div><-- Clearing div ensures the row wraps around the thumbnails -->
</div>
