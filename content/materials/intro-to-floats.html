Introduction to Floats

What is a Float?

A float is a box that is shifted to the left or right on the current line. The most interesting characteristic of a float (or “floated” or “floating” box) is that content may flow along its side (or be prohibited from doing so by the “clear” property). Content flows down the right side of a left-floated box and down the left side of a right-floated box.

Normally, an HTML tag such as a p or div tag would render on its own line.

Example

This is paragraph tag 1

This is paragraph tag 2

Sometimes, we want one paragraph or div tag to render to the right or left of another, with any subsequent content flowing around it.
This is commonly used for images in a content area.
The css property float can be used to push an HTML tag to the right or the left. The possible values are right, left or none (to remove an existing float).

Example

CSS

.image-right {
	float: right;
	margin-left: 20px;
	margin-bottom: 20px;
}

Effects of applying a float

Applying a float has the following effects

  1. The tag will be moved to the right or left of its current position.
  2. Any subsequent tag will wrap around the tag
  3. The floated tag's width will be the maximum size needed to contain whatever elements are inside of it. (If it contains a 600 pixel image, it will be 600 pixels, if it contains text, it will be as wide as is needed to contain the text on a single line).
  4. The floated tag will no longer be contained by any parent elements which are not either floated or having overflow: hidden. This will be important later when we discuss floats for layout.

Because of item #3, it is often necessary to declare a width along with a float. This is particularly the case if we want a caption for our image. Take a look at what happens when you have a div tag floated right and it contains an image and a long caption.

Example: Image with a Long Caption + Width

CSS

.image-right-caption {
	float: right;
	width: 200px;
	margin-left: 20px;
	margin-bottom: 20px;
}
