Introduction to Javascript

What is Javascript

Javascript is a client-side[1] programming language[2] developed by Netscape to facilitate the addition of interactive components to web sites.

  1. Client-side: The "client" in question is web browser (Chrome, Firefox, IE, etc.), which is the software requesting the website from the server, which is where the website files actually live. This is why we can write javascript in an HTML file on our local computer and it will work when we open it in our browser, because the browser (or client) evaluates and executes the javascript.
  2. Programming Language: An artificial language for communicating instructions to a machine/computer (in this case, the browser).

Javascript with HTML/CSS

  • HTML is used for describing our content (this is a paragraph, this is an image, etc.)
  • CSS is used for styling the display of that content.
  • Javascript can add interactivity to the content and manipulate it by adding/remove HTML tags, changing content in addition to many other ways.

As a programming language, Javascript is capable of responding to user input (the value of a text field called "first_name") or the environment (which browser is being used, what the width of the browser is) to cause the site behave differently. Javascript is a tool for attaching behavior to an otherwise static medium of HTML/CSS. Think of it as a layer sitting on top of an HTML/CSS website which provides interactivity such as popup windows, tabbed interfaces, slideshows, animation and more. If you are familiar with Flash/ActionScript, Javascript is like Actionscript for HTML/CSS, it allows you grab certain html elements and manipulate them in any way we like, changing dimensions, positioning, visibility, content and more.

Whenever you are going to write javascript in an HTML document, you need to ensure you put it inside of the special HTML tag <script>.

<script type="text/javascript">
// Javascript goes here, any line preceeded by two forward slashes (//) is a comment and is not actually executed as code
</script>

Alternately, you can place it in a separate file with a .js extensiona and link it using the script tag and src attribute.

<script type="text/javascript" src="something.js"></script>

The Javascript Console

In Chrome, and in the Firebug extension for Firefox, there is a Javascript Console, which allows you to type in javascript code and see the result immediately. Open Chrome and hit F12 to open the Developer console. Click on the Console tab to view the console. Type the following into the console and hit Enter.

alert('hello world')

The Basics: Variables

Programming languages are able to represent and manipulate information, for example they can add 2 numbers together or take sentence and break it apart into words by splitting up on every space. You can store information so you can refer to it later by using a variable by selecting a meaningful name, prefixing it with the 'var' keyword and assigning it a value.

Strings

There are different types of variables in Javascript. The one we see below is an example of a string variable. A string contains text and must be surrounded by quotation marks (single or double). 

<script type="text/javascript">
// Set the myName variable equal to the value of 'Homer Simpson'
var myName = 'Homer Simpson'; 
// Show the value of the myName variable in a popup in the browser
alert(myName); 
</script>

Numbers

There are also numeric types, such as an integer (a whole number, e.g. 5) or float (a decimal number, e.g. 3.1452334). These are not surrounded by quotes.

<script type="text/javascript">
// Set the myHourlyRate variable (its a float)
var myHourlyRate = 15.75;
// Set the hoursWorked variable (its an integer)
var myHoursWorked = 8;
</script>

Javascript can perform mathematical calculations on integers/floats, so if we want to know what my paycheck will be with that hourly rate and hours worked, we can multiple them together using the * symbol. Other symbols that can be used for operations are + (addition), - (subtraction) and / (division). 

<script type="text/javascript">
var myHourlyRate = 15.75;
var myHoursWorked = 8;
alert(myHourlyRate * myHoursWorked);
</script>

Boolean (True or False)

There is a special type of value called a boolean which represents true or false, which are the possible values.

<script type="text/javascript">
var isMale = true;
var isFemale = false;
</script>

Arrays (Lists of Values)

A variable can also hold a list of multiple values, called an Array. The array below contains multiple strings. An individual value of an array can be accessed using the format array[number] where number is the number of the item in the list minus one (the first item is array[0]).

<script type="text/javascript">
var simpsons = [
	'Homer',
	'Marge',
	'Bart',
	'Lisa'
	'Maggie'
];
alert(simpsons[0]);
</script>

Objects (Key/Value Pairs)

Another type of variable in javascript in an object. An object is a selection of key/value pairs and is often used as a set of options. Individual values can be accessed in the format object.key or object[key].

<script type="text/javascript">
var simpsons = {
	'dad': 'Homer',
	'mom': 'Marge'
};
alert(simpsons.dad);
</script>

Functions

The most complex of the types available in javascript is a function. A function is a piece of code that takes some kind of input variables and returns an output variable. For example, you might have a function called add which takes two input variables (x and y) and adds them together, returning the value. The 'alert' function we have been using takes the value of the input you pass in (called an argument) and displays it in a pop up windows.

<script type="text/javascript">
var add = function(x, y) {
	return x + y;
};
alert(add(5, 2));
</script>

Logic: The if/else statement.

We can make our javascript behave differently depending on the values we provied using the if/else statement. We can compare values using some of the comparison symbols (===, >, >=, <, <=) We can also send values to the console so we can see them instead of alerting them by calling the function console.log and passing in the value.

<script type="text/javascript">
var isFive = function(a) {
	if (a === 5) {
		return 'a is 5';
	} else if (a > 5) {
		return 'a is more than 5';
	} else {
		return 'a is less than 5';
	}
};
console.log(isFive(5));
console.log(isFive(6));
console.log(isFive(4));
</script>

Introduction to jQuery

jQuery is a library (or collection of utility functions) for javascript that makes it easier to manipulate HTML websites. The first step to using jQuery is to download it from http://www.jquery.com and to add a link to it in the head of your document using the script tag and src attribute. I would recommend you place it in a js folder for your javascript files.

<script type="text/javascript" src="js/jquery.js"></script>

jQuery often uses CSS selectors in order to grab certain HTML tags (just like we do in CSS for applying styles) and then does something to them. This is achieved using the jQuery function, to which you pass a css selector. To grab all paragraphs on the page and store them in a variable called p, we would use the following code.

<script type="text/javascript">
var p = jQuery('p');
console.log(p);
</script>

Obviously just selecting the paragraph tags in and of itself is not very exciting, but logging the result can help tell you whether your CSS selector is returning any HTML elements or not. Lets say we want to hide all of the paragraphs we grabbed, we can use jQuery's build in function hide().

<script type="text/javascript">
var p = jQuery('p');
p.hide();
</script>

In jQuery, we can chain together multiple functions, so if we wanted to add a class to all of the paragraphs before hiding them, it might look like this.

<script type="text/javascript">
var p = jQuery('p');
p.addClass('myClass').hide();
</script>

$ === jQuery

In order to make jQuery code even easier to write, by default the $ symboly is the same as writing jQuery. Therefore any of the above examples could be written using $ in place of jQuery. You will commonly see this in example plugins.

<script type="text/javascript">
var p = $('p');
p.addClass('myClass').hide();
</script>

jQuery Plugins

jQuery is a very popular Javascript library and many people have released extensions for it that provide functionality for tabs, accordions, slideshows and lightboxes. These extensions are called plugins, and generally a google search for 'slideshow jquery plugin' will yield many results. Just like the jquery library itself, the first step to using a plugin is to download it and to link any javascript/css files requried to your HTML page. Generally plugins come with a README file or have a section on the website for installation that tells you what you need to include for the plugin to work. Most jQuery plugins work by extending the functions available once you grab some HTML elements using $('p'). For example, you may find an accordion plugin called fancyAccordion that allows you to call $('p').fancyAccordion() just as we called $('p').hide() above.

jQuery Lightbox Plugin: FancyBox

The FancyBox plugin is a popular jQuery plugin for creating lightbox effects. To begin, download the zip file from the website and place the contents in a js folder. Change the folder name to fancybox. Open the README.md file and go to the How to Use section.
